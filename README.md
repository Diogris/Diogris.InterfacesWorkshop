# Практикум по C#. Интерфейсы
## Описание

Проект содержит небольшую демонстрационную задачу, направленную на отработку базовых навыков работы с интерфейсами и с DI (внедрением зависимостей).

## Структура

### IAddition

Интерфейс, описывающий метод сложения.

### ILogger

Интерфейс, описывающий методы логирования.

### Calculator

Класс, реализующий интерфейс ***IAddition***, в частности метод сложения двух чисел ***Sum***

### Logger

Класс, реализующий интерфейс ***ILogger***, а именно:
- Метод вывода информационного сообщения ***(Info)***;
- Метод вывода сообщения об ошибке ***(Error)***.

### Worker

Класс, содержащий основную логику приложения.
В бесконечном цикле с возможностью выхода по команде ***Ctrl+C*** запрашивает два числа, суммирует их и логгирует процесс.
В случае некорректного ввода числа, выводит сообщение об ошибке и предлагает повторить ввод.
